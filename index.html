Commandos basicos git
git add (file|.)
Este comando pasa un archivo o una lista de archivos del estado untracked a stage
git commit -m "mensaje explicativo las comillas son obligatorias"
Este comando consolida los cambios que esten en el stage
git clone url-del-proyecto
Este comando clona un proyecto del remoto al local
git push origin (nombre-de-la-rama)
Este comando "empuja" los cambios del local al remoto
git pull origin (nombre-de-la-rama)
Este comando "jala" los cambios del remoto al local
git remote origin (https-url|ssh-url)
Este comando establece una conexion entre el reposiotorio local y el reposiotorio remoto
git remote remove origin
Este comando se usa para remover la conexion existente entre el reposiotorio local y el reposiotorio remoto
git reset HEAD (file|.)
Este comando pasa los cambios del stage al untraked
